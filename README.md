# bobodoctor

## 專案說明

此專案使用 Python 的 request 模組，爬取波波醫生名單。 使用selenium打開google我的地圖，將爬取到的名單製作成地標放入google我的地圖。

## 技術關鍵字

- Selenium
- Time
- requests
- BeautifulSoup
- undetected_chromedriver

## 功能

- 爬取波波醫生名單
- 製作地標地圖

## 使用資料

- 波蘭醫師一覽表
- google我的地圖

## 環境需求

- Python 3.9.13
- Selenium 4.24.0
- request 2.32.3
- beautifulsoup4 4.12.3
- 3.5.5
